GIT

upload a new project to git

git init                                  - creates .git  
git add *.c
git commit -m 'remark'

import an existing repository

git clone https://github.com/yaniv33/MyWebSite.git

 

git fetch origin               - really only downloads new data from a remote repository

git pull origin master   - your current HEAD branch with the latest changes

 

register your user and email

git config –global user.name "Yaniv"

git config –global user.name "yaniv33@gmail.com"

 

 

git status                              - returns the status of the git folder  
git log

git checkout    [commit hash] – commit is taken from log, same as rollback

                        | [branch] – moves from one branch to another

 

git branch ?[name of branch] – create branch 

main branch is master
