Starting with React
  1. public\index.html calls to scripts and <div id="root">
  2. index js creates element root and calls App
    const root = ReactDOM.createRoot(document.getElementById('root'));
root.render(
  <React.StrictMode>
    <App />
  </React.StrictMode>
);
  1. App.js creates function App that returns the html of the site
function App() {
  return (
...
  );
}
export default App;
  1. passing a class is with 
      <TodoApp/>

      <TodoApp element={<PokedexHeader name={"Greg"} author={"Yaniv"}/>}/>
in package.json set the scripts to start the app as 
"scripts": {
  "start": "react-scripts start",
  "start-pc": "set PORT=3021&& react-scripts start",
