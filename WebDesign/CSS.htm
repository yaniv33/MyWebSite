Features

More info here

.nav-list        – applies to nav-list
.nav-list li     - applies to all li tags under nav-list
.columns > *     - apply to all children of the tag that receives class="columns"
div > p          - applies <p> where parent is div
div + p          – applies to the <p> place immediately after div 
p ~ ul           - applies to <ul> that before it comes <p>
.columns:nth-child(2)   - apply to the second child of the tag that receives class="columns"
p:nth-child(odd)        - apply to add children (could also be with : even / 3n+1 and so on …
#firstname      - apply to element with <tag id=”firstname”>
p.intro         – apply to element <p class=”intro”>

[target]          – apply to all elements with target such as <p target=””>
[target=flower]   – apply to all elements with <p target=”flower”>
[title=~flower]   – apply to all elements with title containing  <p title=”flower sead”> (~ contains , ^ begins, $ ends, )
a[href$=”otot”]   - apply to <a href=”http://myotot.ws>

input:checked         – apply to all checked input
input[type=checkbox] – apply type checkbox (could radio , range) … set color as => accent-color: rgb(0, 0, 255); 

:not(p) – apply to none p elements

Units
em 	Relative to the font-size of the element (2em means 2 times the size of the current font) 	
ex 	Relative to the x-height of the current font (rarely used) 	
ch 	Relative to the width of the "0" (zero) 	
rem 	Relative to font-size of the root element 	
vw 	Relative to 1% of the width of the viewport* 	
vh 	Relative to 1% of the height of the viewport* 	
vmin 	Relative to 1% of viewport's* smaller dimension 	
vmax 	Relative to 1% of viewport's* larger dimension 	
% 	Relative to the parent element
