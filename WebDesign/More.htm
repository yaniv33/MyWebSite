Grid vs Flex

Grid

use if needs fixed columns length

does not deal well with long data

control is from the parent 

grid-container {
    grid-template-columns: repeat(4, 1fr);
    grid-template-columns: repeat(auto-fit, minmax(300px, 1fr));
}



Flex

used mostly for flex (each component takes as much and needed)

needs to impact children to get the full effect

flex-container{
    display: flex
}
flex-container > * {
    flex-grow:1;
    flex-basis: 33%
}
