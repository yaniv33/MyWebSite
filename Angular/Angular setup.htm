Install node

install node.js from https://nodejs.org/en/
run in cmd > npm install -g @angular/cli

check version

FIND VERSION angular : ng --version            OR look at file package.json
UPDATE VERSION :    ng update @angular/cli @angular/core

FIND VERSION node : node -v
UPDATE VERSION :     npm install -g node@latest        or         install from site (https://nodejs.org/download/release/)  

FIND VERSION npm  : npm version
UPDATE VERSION :        npm install -g npm@latest

FIND VERSION typescript  : npm version
UPDATE VERSION :       npm install typescript@latest

files location

    f:\Program Files\nodejs\
    %AppData%\Roaming\npm

angular working order

main.js > main module > routes adn components > children

file structure

package.json - holds shortcuts for scripts and dependencies
angular.json - holds the styles and assest

Deploy angular

add lines to angular.json
      "baseHref" : "/systemhandler/",
      "deployUrl": "/systemhandler/",
      "outputPath": "systemhandler",

run > ng build (or ng build --prod)

upload the folder to  WEB-APP
